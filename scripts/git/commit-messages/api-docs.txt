## 구현 내용

### OpenAPI 3.0 스펙 정의
- src/lib/swagger.ts: 완전한 API 스펙
- 모든 엔드포인트 문서화
- 스키마 정의 (JobOrderData, JobOrderOptions, etc.)
- Request/Response 예시 포함

### Swagger UI 구현
- /api-docs: 전통적인 Swagger UI
- Dynamic import로 SSR 이슈 해결
- Console.error 필터링으로 경고 억제
- "Try it out" 기능으로 실시간 테스트

### Scalar API Reference 구현
- /api-docs-scalar: 현대적인 API 문서
- React 19 완벽 호환 (경고 없음)
- 다크 모드 지원
- 다양한 언어 코드 예시
- 향상된 검색 기능

### Swagger JSON 엔드포인트
- GET /api/swagger: OpenAPI 스펙 제공
- Postman/Insomnia import 가능
- 자동 생성된 스펙

## 주요 기능

### Swagger UI
- 업계 표준 인터페이스

### Scalar API Reference
