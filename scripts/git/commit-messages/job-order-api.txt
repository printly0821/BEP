## 구현 내용

### RESTful API
- GET /api/job-orders: 제작 의뢰서 목록 조회
- GET /api/job-orders/:id: 제작 의뢰서 상세 조회
- POST /api/job-orders/:id: 제작 의뢰서 생성 (테스트용)
- Mock Database: 3개 샘플 데이터 포함

### TypeScript 타입 정의
- JobOrderData: 제작 의뢰서 전체 데이터
- JobOrderOptions: 옵션 정보 (7개 필드)
- JobOrderApiResponse: API 응답 형식
- JobOrderSummary: 목록 조회용 요약 데이터

### 제작 의뢰서 UI
- JobOrderReportView: PDF 레이아웃 기반 리포트 뷰
- 실제 1D 바코드 생성 (react-barcode)
- 실제 QR 코드 생성 (react-qr-code)
- 주문번호/문서번호 자동 바코드화

### 동적 리포트 생성
- API에서 JSON 데이터 fetch
- 드롭다운으로 제작 의뢰서 선택
- 실시간 데이터 렌더링
- PDF 다운로드 지원

### 테스트 페이지
- /test-report: API 통합 테스트 페이지
- 목록 선택 UI
- 로딩 상태 표시
- 에러 핸들링
- PDF 다운로드 버튼

## 기술 스택
- Next.js 15 App Router
- TypeScript
- react-barcode v1.6.1
- react-qr-code v2.0.18
