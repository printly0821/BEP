## Description

# 요구사항
- 화면: 프로젝트 목록 페이지, 상세 패널 또는 모달, 새 프로젝트 생성, 이름 변경, 삭제
- Import/Export 진입 버튼 제공: 엑셀 Import, PDF/엑셀 Export 접근성 향상
- 반응형 및 접근성 준수, 로딩/빈 상태 UX, 에러 토스트
- 다국어 레디: 텍스트 키 분리

# Implementation details
- Next.js 15 App Router로 /projects, /projects/[id] 구성, 상세는 Drawer 모달로 표시
- UI: shadcn-ui DataTable, Dialog, Drawer, Button, Input, DropdownMenu 사용
- 데이터: React Query로 목록/상세 연동, 스켈레톤 플레이스홀더
- 상호작용: 생성 다이얼로그에서 이름 입력 후 저장 API 호출, 삭제는 Confirm 모달
- Import/Export: 헤더 액션바에 버튼 배치, 모바일에서는 Bottom Sheet로 배치
